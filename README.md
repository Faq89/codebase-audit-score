# Codebase Audit Score

Enterprise-grade auditing skill for AI coding assistants.

## Features
- Full repository discovery
- Architecture, Security, UX/UI, Database and Performance audits
- Weighted scoring
- Prioritized roadmap

## Installation

Copy `SKILL.md` into your AI skill directory.

## Usage

Ask:

```
Run Codebase Audit Score against this repository.
```
