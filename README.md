# Codebase Audit Score 📊

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Enterprise-grade auditing skill for AI coding assistants. This skill performs a comprehensive analysis of software projects, producing weighted engineering scores, actionable recommendations, and prioritized roadmaps.

*Read this in other languages: [Español](#español-manual-en-español)*

## 🇬🇧 English Manual

### Features
- **Full repository discovery:** Understands the entire codebase context.
- **Comprehensive audits:** Architecture, Security, UX/UI, Database, and Performance.
- **Weighted scoring:** Enterprise-level metrics and health indicators.
- **Prioritized roadmap:** Actionable next steps based on audit results.

### Installation
Copy `SKILL.md` into your AI assistant's skill or plugin directory.

### Usage
Ask your AI assistant:
> "Run Codebase Audit Score against this repository."

### Scoring Weights

| Category | Weight |
|---|---:|
| Architecture | 20% |
| Security | 20% |
| Code Quality | 15% |
| Performance | 10% |
| Database | 10% |
| UX | 10% |
| UI | 5% |
| Accessibility | 5% |
| Scalability | 5% |
